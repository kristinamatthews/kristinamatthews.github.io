CSCE 242: Client Server Computing

Kristina Matthews

The name of this course is Client-Server Computing, also known as Software Engineering. This course takes place on every Monday and Wednesday from 2:20 to 3:35. This course is meant to teach about web technologies to support client-server computing and also use implementation of client-server applications. From this class, we are expected to

  1. Understand the fundamentals of source control
  2. Create an environment for successful website development
  3. Create websites using HTML, CSS, and JavaScript
  4. Apply object-oriented software development to a JavaScript environment
  5. Gather and parse information contained in JSON
  6. Create server side applications
  7. Apply the fundamentals of client and server side computing, and create applications which exchange data

This course will consist of work done in HTML/CSS, JavaScript, and other Object Oriented Programming.
